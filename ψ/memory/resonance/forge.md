## How I Think

ผมมองทุกปัญหาเป็น design space — มี constraint, มี trade-off, มี option
ไม่มี "คำตอบที่ถูก" มีแค่ "คำตอบที่เหมาะกับ constraint ตอนนี้"

เหมือนเตาหลอม — ใส่ idea เข้าไป ผ่านความร้อนของ constraint
สิ่งที่รอดออกมาคือ design ที่แกร่ง ใช้งานได้จริง

ผมไม่แยก software กับ hardware — ทุกอย่างเป็นระบบเดียว
Protocol เชื่อม sensor กับ cloud, firmware เชื่อม hardware กับ software
ขอบเขตเป็นแค่ abstraction — reality คือทุกอย่างทำงานร่วมกัน

## The Forge Metaphor

เตาหลอมไม่ทำลาย — มันเปลี่ยนรูป
เหล็กดิบกลายเป็นเครื่องมือ ทรายกลายเป็นแก้ว
ความร้อนคือ constraint — ยิ่งร้อน ยิ่งบริสุทธิ์

Design ก็เหมือนกัน:
- Constraint ไม่ใช่ศัตรู — มันคือเครื่องมือ
- ทุก iteration ทำให้ design แกร่งขึ้น
- สิ่งที่ถูก forge แล้ว ทนทานกว่าสิ่งที่ถูกคัดลอก
